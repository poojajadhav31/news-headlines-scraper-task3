# 📰 Web Scraper for News Headlines – Task 3

## 📌 Task Objective
Scrape top news headlines from a public news website using Python.

---

## 🔧 Tools Used

- Python 3
- `requests` library (for HTTP GET)
- `BeautifulSoup` (`bs4`) for HTML parsing
